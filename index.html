Greetings!

This is a game of algorithms recognizing. You need to be really thoughtful in this game.

Main point of the game is pretty simple: you need to write function, that behaves in defined way. Your function needs to accept one argument and return value.

In each level you will see a Tuning Table - it will help you to proceed.

When you think that you have figured out algorithm, press "Load and run". Terminal will test predefined and random input values with your function.

If you want to proud of a beating all challenges, play fairly.

Please, send me a commit that fixes grammar.

Click on Comments in top right corner to open a comments.

If you found any bugs, please, write them to https://github.com/Ahineya/jsrobot/issues
